<task type="auto">
  <name>Task 1: Add download button and .xlsx export</name>
  <files>index.html</files>
  <action>
1. Add a download button in the center panel header (next to the badge):
   - Use a download/arrow-down SVG icon
   - Style to match existing panel header aesthetic (muted by default, brand-400 on hover)
   - Button hidden by default (display:none), shown only when a file is open
   - Add CSS class `panel-header-action` for the button styling

2. In the Excel editor IIFE (created in Plan 01), add the download/export logic:
   - Get reference to the download button element
   - On click:
     a. Get current data from Jspreadsheet: `instance.getData()` (returns array-of-arrays)
     b. Create a new SheetJS workbook: `XLSX.utils.book_new()`
     c. Convert data to worksheet: `XLSX.utils.aoa_to_sheet(data)`
     d. Append worksheet to workbook: `XLSX.utils.book_append_sheet(wb, ws, 'Sheet1')`
     e. Write to binary: `XLSX.write(wb, {bookType:'xlsx', type:'array'})`
     f. Create Blob and trigger download using an anchor element with `URL.createObjectURL`
     g. Use original file name for the download (from `window._excelState.fileName`), or 'export.xlsx' as fallback

3. Show the download button when a file is opened (in the openFile function from Plan 01).
   Hide it when returning to empty state.

4. Add hover/focus/active states on the download button per project guardrails (transform + opacity only).
  </action>
