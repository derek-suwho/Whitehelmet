<task type="auto">
  <name>Task 1: Wire spreadsheet library with click-to-open and cell editing</name>
  <files>index.html</files>
  <action>
1. Add CDN script tags in `<head>` for:
   - SheetJS (xlsx): `https://cdn.sheetjs.com/xlsx-0.20.3/package/dist/xlsx.full.min.js` (parses/writes .xlsx)
   - Jspreadsheet CE v5: `https://cdn.jsdelivr.net/npm/jspreadsheet-ce@5/dist/index.min.js` + CSS `https://cdn.jsdelivr.net/npm/jspreadsheet-ce@5/dist/jspreadsheet.min.css`
   - jSuites (required dependency for Jspreadsheet CE): `https://jsuites.net/v5/jsuites.js` + CSS `https://jsuites.net/v5/jsuites.css`

2. Create a new IIFE for the Excel editor (after the file ingestion IIFE). This IIFE must:
   - Get references to center panel body and header badge
   - Expose a global `window.openFile(fileObj)` function that:
     a. Reads the File object using FileReader as ArrayBuffer
     b. Parses with `XLSX.read(data, {type:'array'})` to get workbook
     c. Converts first sheet to array-of-arrays via `XLSX.utils.sheet_to_json(ws, {header:1})`
     d. Clears the center panel body (remove empty state)
     e. Creates a Jspreadsheet CE v5 instance with the parsed data. v5 uses `worksheets` array:
        ```javascript
        jspreadsheet(container, {
          worksheets: [{
            data: arrayOfArrays,
            minDimensions: [10, 20],
            tableOverflow: true,
            tableWidth: '100%',
            tableHeight: (panelBody height - 8) + 'px',
            columns: autoGeneratedFromDataWidth,  // default width 120px each
            allowInsertRow: true,
            allowInsertColumn: true
          }]
        });
        ```
        - Returns an array of worksheet instances — store `instance[0]` for data access
        - Full cell editing enabled by default
     f. Updates the header badge to show the file name (truncated to 20 chars)
     g. Stores the current jspreadsheet worksheet instance (`instance[0]`) and workbook reference on `window._excelState` for Plan 02 to access

3. Modify the file ingestion IIFE to add click handlers on source file items:
   - In `createFileItem()`: add a click event on the item div that calls `window.openFile(f.file)` where `f.file` is the raw File object
   - In `createFolderItem()`: add click events on the child file divs (inside the folder children container) that call `window.openFile(f.file)`
   - IMPORTANT: The folder header click already toggles expand/collapse. Only add openFile click on the child file items inside folders, not the folder header itself.
   - Add visual feedback: clicked file gets a selected/active style (e.g., background highlight). Store selected file reference. Remove highlight from previous selection.

4. Add CSS styles for the spreadsheet integration:
   - Override Jspreadsheet default styles to match the dark theme (surface-panel bg, content-primary text, surface-border borders)
   - Selected cell highlight using brand-400 color
   - Style the jspreadsheet container to fill the center panel body
   - Hide the Jspreadsheet default toolbar (we don't need it for v1)
   - Ensure scroll works properly within the fixed-height center panel

5. Keep the empty state ("Select a source file to view") visible when no file is open. Only replace it when a file is clicked.
