    Append the following code block inside the existing &lt;script&gt; block, AFTER the closing `})();` of the existing chat IIFE (at approximately line 737, before the closing &lt;/script&gt; tag):

    ```javascript
    // Role switcher — sets data-role on body, CSS handles panel visibility
    (function () {
      var roleSelect = document.getElementById('roleSwitcher');

      function applyRole(role) {
        document.body.dataset.role = role;
      }

      roleSelect.addEventListener('change', function () {
        applyRole(this.value);
      });

      // Apply default role on load (contractor = show everything)
      applyRole(roleSelect.value);
    })();
    ```

    Keep the code wrapped in an IIFE to match the existing script style. Use `var` for consistency with the existing chat script. Do NOT use `textContent` or `innerHTML` with the role value — only assign to `dataset.role`, which is safe.
