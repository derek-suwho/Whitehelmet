Three rules that govern this architecture
The frontend never calls the AI directly — all AI requests go through Express, which holds the key server-side.
The frontend never queries the database — all data access goes through Express API routes.
API contracts are the handoff point — Groups 1 & 2 build against agreed request/response shapes while Group 3 builds the real endpoints.
