## Description

Variant of Project Home where the filter panel is active. The filter icon (top-right of structure table) is highlighted/active.

## Layout

Same as Project Home — With Phase, with filter overlay or panel active.

- Phase selector, "Show Cores Only" toggle visible
- Filter icon button appears highlighted (active state)
- Structure table filtered based on active filter criteria

## Filter Criteria (inferred from design)

- Zone filter
- Core/Plan type filter
- Completion status filter (Completed / In Progress)
- "Show Cores Only" toggle (always visible, outside filter panel)
