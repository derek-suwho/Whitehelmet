## Visual Notes

- Icon is a hexagonal geometric shape in **yellow-green** (`#C8D400` approx) with a grey accent
- Wordmark uses a clean sans-serif in dark charcoal on light, white on dark
- Dark variants shown on solid black backgrounds
- Icon dimensions: ~70×80px in Figma at design scale

## Usage

- Use `Variant=Logo` as the default app header logo
- Use `Variant=Icon` for favicon / collapsed sidebar states
- Use `Dark=True` variants on the dark navy top navbar (`#1a1a2e` approx)
- Never stretch or recolor — use the symbol as-is
